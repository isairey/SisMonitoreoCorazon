# 🤖 AI Studio App

Aplicación desarrollada con tecnologías modernas para la ejecución y despliegue de soluciones basadas en **Inteligencia Artificial**, integrando APIs avanzadas para generación y procesamiento de contenido.

</div>

---

## 📌 Descripción

Este proyecto contiene todo lo necesario para ejecutar una aplicación de **AI Studio** en entorno local, permitiendo experimentar, desarrollar y desplegar funcionalidades basadas en modelos de inteligencia artificial.

La aplicación está diseñada para ser rápida, modular y fácil de configurar.

---

## 🚀 Características

- ⚡ Ejecución local rápida  
- 🤖 Integración con APIs de IA (Gemini)  
- 🧩 Arquitectura modular  
- 🌐 Entorno listo para desarrollo  
- 🔐 Manejo seguro de variables de entorno  

---

## 🛠️ Tecnologías utilizadas

- **Node.js** → Entorno de ejecución  
- **JavaScript / TypeScript** → Lógica de aplicación  
- **AI Studio** → Plataforma base  
- **Gemini API** → Procesamiento de IA

## ⚙️ Requisitos

Antes de comenzar, asegúrate de tener instalado:

- Node.js (versión 16 o superior recomendada)  
- npm o yarn  
- Clave de API de Gemini  

---

## 🔧 Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/tuusuario/ai-studio-app.git
```
Acceder al proyecto:
```
cd ai-studio-app
```
Instalar dependencias:
```
npm install
```
---

## 🔐 Configuración

Crear un archivo .env.local en la raíz del proyecto y agregar:
```
GEMINI_API_KEY=tu_api_key_aqui
```
---
## ▶️ Ejecución local
```
npm run dev
```
Abrir en el navegador:
```
http://localhost:3000
```
